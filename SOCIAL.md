## 🏗️ Deploy in 30 Seconds

```bash
git clone https://github.com/intro0520/Tg-Fake.git
cd Tg-Faka && pip install -r requirements.txt
python -m uvicorn web:app --port 8000
```

Done. Your admin dashboard is live:
🌐 `http://your-server:8000`

Domain + Nginx + HTTPS? We have a full step-by-step guide.
